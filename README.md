# MeetMind AI 🚀

MeetMind AI is an AI-powered meeting assistant that helps users prepare for meetings by analyzing documents, generating meeting strategies, creating executive briefs, answering context-aware questions using Retrieval-Augmented Generation (RAG), and automating task management through Todoist integration.

## ✨ Features

- 📄 Analyze PDF and TXT documents for meeting context
- 🤖 Generate AI-powered meeting strategies and executive briefs
- 💬 Ask context-aware questions using RAG
- 👥 Multi-agent workflow powered by CrewAI
- 📋 Extract meeting tasks and create them in Todoist
- 📢 Send task notifications through Telegram
- 🌐 Interactive Streamlit web application

## 🛠️ Tech Stack

- Python
- Streamlit
- Google Gemini
- LangChain
- CrewAI
- FAISS Vector Database
- Todoist REST API
- Telegram Bot API

## 🚀 Workflow

1. Configure meeting details.
2. Upload meeting documents (optional).
3. AI analyzes the meeting context.
4. Generates meeting strategy and executive brief.
5. Ask questions using the built-in AI assistant.
6. Extract action items and create Todoist tasks automatically.
